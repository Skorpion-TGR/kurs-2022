Урок №2

Тема: анатомія HTML - теги та їх різновиди

Теги HTML поділяються на так звані будівельні теги та теги для контенту

  • Будівельні теги

    Будівельні теги застосовують для побудови структури сайту, каркасу майбутнього сайту
    наприклад як будівництво каркасу будинку

    та подальшого їх заповнення тегами для контенту

    До будівельних тегів відносять:

    • тег div - тег, без конкретного призначення та може застосовуватись не обмежену кількість разів
    Теги з конкретним семантичним призначенням:
    • тег header - тег, який використовують для визначення верхньої частини сайту (або блоку) - "шапка сайту"
      бажано застосувати лише один раз
    • тег main - тег, яким визначають головну (унікальну) частину сайту (основне наповнення сайту)
      застосовується виключно один раз
    • тег footer - тег, який визначає нижню частину сайту, "дно сайту"
      також застосовується виключно один раз
    • тег nav - тег, який застосовують для створення навігаціїї
      може застосовуватись багатократно
  • Теги для контенту

    Ці теги застосовують для наповнення контентом каркасу сайта, створений будівельними тегами
    наприкладі будинку - монтування фасаду

    До контентних тегів відносять:

    • теги h1-h6 - теги призначені для заголовків
      тег h1 - застосовується лише один раз та визначає найголовніший заголовок сайту
      теги h2-h6 - можуть застосовуватись багаторазово
    • теги mark та span - строкові теги, які застосовують для маркування та визначення рядків відповідно
      тег mark - виділяє текст жовти кольором
      тег span - визначає рядок
    • тег р - застосовують для визначення параграфа
    • тег br - застосовують для переносу тексту на новий рядок
    • теги ul/ol та li - теги для визначення списків

      тег ul - застосовують для маркованого списку
      тегol - для нумерованого списку

      тег li - являється невід'ємною частиною тегів ul/ol, яким визначають пункти списку
      даний тег не використовують окремо від тегів ul/ol так само як і вони не використовуються без тегу lі

    • тег а - строковий тег для визначення посилання на сторонній сайт або на певний блок поточної сторінки
    • тег img - тег для застосування зображення на сторінці
      даний тег специфічний, тому що має строково-блочні властивості тегів
